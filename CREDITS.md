## المساهمون

نرحب بالمساهمات! إذا أردت تحسين الأداة أو إضافة ميزة جديدة، يرجى:

1. عمل Fork للمستودع.
2. إنشاء فرع جديد (`git checkout -b feature/amazing-feature`).
3. رفع التغييرات (`git commit -m 'Add some amazing feature'`).
4. Push إلى الفرع (`git push origin feature/amazing-feature`).
5. فتح Pull Request.
